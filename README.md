# House Price Prediction

## Project Overview
This project focuses on predicting house prices using Machine Learning techniques. The dataset was cleaned, analyzed, and used to train predictive models that estimate house prices based on various property features.

## Objectives
- Load and explore housing data
- Clean and preprocess the dataset
- Convert categorical variables into numerical format
- Train and evaluate machine learning models
- Visualize important patterns in the data
- Identify factors that influence house prices

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Machine Learning Models
1. Linear Regression
2. Random Forest Regressor

## Visualizations
- House Price Distribution Histogram
- Correlation Heatmap
- Actual vs Predicted Price Scatter Plot

## Results
The Random Forest Regressor performed better than Linear Regression and produced more accurate predictions. Features such as area, number of bathrooms, air conditioning, and location had a strong influence on house prices.
